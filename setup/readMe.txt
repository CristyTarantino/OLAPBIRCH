I dataset sono scaricabili da

http://si.deis.unical.it/%7Ecuzzocrea/tpch10mb.sql
http://si.deis.unical.it/%7Ecuzzocrea/tcph1gb.sql
http://si.deis.unical.it/%7Ecuzzocrea/tcph2gb.sql


Dopo averli caricati in MySQL, editare il file "config.ini" con le impostazioni desiderate
e i dati di accesso al database.
Impostare la profondità massima nel file XML "tpch.xml"
In seguito eseguire lo script "script_mod.sql".

Ora si puo' avviare il programma da "avvia.bat" o da "avvia.sh"
